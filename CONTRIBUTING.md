# Contributing

Thanks for your interest in improving this project.

- **Code and templates:** See the full guide in [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md) (repository layout, adding backends/frontends, contract changes, compliance tests, and pull request expectations).
- **Issues:** Use GitHub Issues for bugs and feature ideas; avoid filing undisclosed security issues there (see [SECURITY.md](SECURITY.md)).
- **License:** By contributing, you agree that your contributions are licensed under the same terms as the project ([LICENSE](LICENSE), MIT).

If you are only consuming the starter or the `create-saas-app` CLI, you do not need to read the contributor guide unless you plan to extend templates or upstream changes.
