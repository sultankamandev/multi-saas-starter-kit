# Security

## Supported versions

We aim to support the latest published major version of this repository and the `create-saas-app` CLI on npm. Security fixes are applied to the default branch when practical.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for undisclosed security problems.

- Prefer [GitHub Security Advisories](https://github.com/your-org/saas-starter-kit/security/advisories/new) for this repository (replace `your-org/saas-starter-kit` with your actual path after you publish).
- If that is not available, contact the maintainers privately with a clear description, affected component, and steps to reproduce.

We do not guarantee a response timeline or SLA. Thank you for helping keep users safe.
